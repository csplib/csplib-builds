A book ( [Gorgon92Battleships] ) of puzzles has been published.

Fathom It! \cite{FathomIt} is a shareware program for playing and solving these puzzles. This gives you a program to let you solve puzzles on screen, as well as a rule-based solver. The trial version contains 80 puzzles: 10 each in four levels of difficulty and two sizes (7x7 and 10x10.) The full version contains thousands. All puzzles distributed can be solved by the rule based solver without backtracking. However puzzles exist that it cannot solve (see the overview page).
